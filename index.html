twilio's PEAR Channel Server

Registering the channel: 
pear channel-discover twilio.github.com/pear
Listing available packages: 
pear remote-list -c twilio
Installing a package: 
pear install twilio/package_name
Installing a specific version/stability: 
pear install twilio/package_name-1.0.0
pear install twilio/package_name-beta
Receiving updates via a feed: 
http://twilio.github.com/pear/feed.xml

Services_Twilio - PHP helper library for Twilio

A SDK (or helper library, as we're calling them) for PHP developers to write applications against Twilio's REST API and generate TwiML responses.

Install commandpear install twilio/Services_Twilio
